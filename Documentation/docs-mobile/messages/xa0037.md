# .NET for Android error XA0037

## Issue

The `PrepareAbiItems` build task was given a `Mode` value that it does
not recognize. This is an internal error: the `Mode` is provided by the
.NET for Android MSBuild targets, so this should not occur in a normal
build.

## Solution

This error generally indicates a bug in .NET for Android. Please report
an issue at https://github.com/dotnet/android/issues and include the
build log.

## Example messages

```
error XA0037: Unknown mode: SomeUnknownMode
```
