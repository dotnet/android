# .NET for Android warning XA0146

## Example messages

> warning XA0146: Attempt to check whether 'path/to/libexample.so' is a correctly aligned ELF file failed with exception, ignoring alignment check for the file.

## Issue

An exception occurred while trying to read a native shared library to
verify its ELF segment alignment. The alignment check has been skipped
for this file.

## Solution

Ensure the file is a valid ELF shared library and that the build process
has the necessary permissions to read it. If the file is not a native
library, it may be incorrectly included in the build.
