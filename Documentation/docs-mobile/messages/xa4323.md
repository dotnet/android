# .NET for Android warning XA4323

## Example messages

```
warning XA4323: Ignoring directory 'obj/Debug/net11.0-android/res/layout-v31' as it does not exist!
```

## Issue

A resource directory passed to `aapt2 compile --dir` no longer exists on disk.
This can happen when stale intermediate item metadata points at a directory that
was deleted or moved between builds.

## Solution

Ensure the referenced resource directory exists, or remove stale build
intermediate files (`bin/` and `obj/`) and rebuild.
