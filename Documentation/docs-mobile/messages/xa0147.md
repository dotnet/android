# .NET for Android warning XA0147

## Example messages

> warning XA0147: Attempt to check whether 'path/to/libexample.so' is a valid ELF file failed with exception, ignoring AOT check for the file.

## Issue

An exception occurred while trying to read a native shared library to
determine whether it is an empty AOT library. The AOT check has been
skipped for this file.

## Solution

Ensure the file is a valid ELF shared library and that the build process
has the necessary permissions to read it. If the file is not a native
library, it may be incorrectly included in the build.
