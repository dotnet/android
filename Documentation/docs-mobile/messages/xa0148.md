# .NET for Android warning XA0148

## Example messages

> warning XA0148: Attempt to check whether 'path/to/libexample.so' is a valid ELF file failed with exception, ignoring symbol 'JNI_OnLoad@.dynsym' check for the file.

## Issue

An exception occurred while trying to read a native shared library to
check for a specific symbol. The symbol check has been skipped for this
file.

## Solution

Ensure the file is a valid ELF shared library and that the build process
has the necessary permissions to read it. If the file is not a native
library, it may be incorrectly included in the build.
