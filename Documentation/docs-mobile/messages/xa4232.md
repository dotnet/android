# .NET for Android error XA4232

## Example messages

```
error XA4232: The Android code generation target 'JavaInterop1' is no longer supported. Remove the '_AndroidJcwCodegenTarget' property or set it to 'XAJavaInterop1'.
```

## Issue

The project sets the internal `_AndroidJcwCodegenTarget` MSBuild property to
`JavaInterop1`. This code generation target is no longer supported for any
.NET for Android runtime.

## Solution

Remove this property assignment from the project:

```xml
<_AndroidJcwCodegenTarget>JavaInterop1</_AndroidJcwCodegenTarget>
```

The property defaults to `XAJavaInterop1`. If a build customization must set
it explicitly, update the value to `XAJavaInterop1`.
